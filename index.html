2022 St. Louis Cardinals Roster

Hey Cardinal Nation!

This free API will allow you to have access to the 2022 roster for your St. Louis Cardinals as well as a breakdown of players by each position. The API includes the names, numbers, and photos of your favorite St. Louis Cardinals player or players.

The endpoints are:

  • /players
  • /players/pitchers
  • /players/catchers
  • /players/infielders
  • /players/outfielders

The information in the API is accurate up through May 10, 2022
All information will be returned in JSON format

/players

The players endpoint is accessed using the Players Endpoint

This route uses the GET method and returns the roster of all the players on the team with their number and a picture.

/players/pitchers

The pitchers endpoint is accessed using the Pitchers Endpoint

This route uses the GET method and returns all of the pitcher on the team with all the previous information.

/players/catchers

The catchers endpoint is accessed using the Catchers Endpoint

This route uses the GET method returns all the catchers on the team with all the previous information.

/players/infielders

The infielders endpoint is accessed using the Infielders Endpoint

This route uses the GET method and returns all the infielders on the team with all the previous information.

/players/outfielders

The outfielders endpoint is accessed using the Outfielders Endpoint

This route uses the GET method and returns all the outfielders on the team with all the previous information.
